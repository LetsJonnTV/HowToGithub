✅ GitHub Projekt Checkliste

🛠️ 1. Projekt auf GitHub erstellen

  • Auf github.com einloggen
  • Rechts oben auf "+" > New repository klicken
  • Repository-Name vergeben
  • Public oder Private wählen
  • Optional: README, .gitignore, LICENSE hinzufügen
  • Auf "Create repository" klicken

🧩 2. Projekt lokal in VS Code einrichten

  • Git installieren
  • VS Code installieren
  • Repository klonen:
git clone https://github.com/benutzername/mein-projekt.git
cd mein-projekt
code .

📦 3. Struktur und Dateien einrichten

  • Projektstruktur (z. B. src/, docs/, tests/)
  • README.md anpassen
  • .gitignore ergänzen
  • Initialer Commit:
git add .
git commit -m "Initial commit"
git push origin main

🧾 4. Issues nutzen

  • Issues-Tab > New Issue
  • Titel + Beschreibung
  • Labels, Milestones setzen
  • Personen zuweisen
  • PR/Commits mit Fixes #1 verknüpfen

📋 5. Projektmanagement mit GitHub Projects

  • Neues Board unter "Projects" anlegen
  • Spalten: To Do, In Progress, Done
  • Issues und PRs als Karten hinzufügen

🔀 6. Branches & Pull Requests

  • Branch erstellen:
git checkout -b feature/mein-feature
git push origin feature/mein-feature
  • PR erstellen, Feedback einholen und mergen

🧪 7. Tests & Codequalität

  • Linter (z. B. ESLint, Prettier) einrichten
  • Tests schreiben und lokal ausführen:
npm test / pytest

🔁 8. Pflege & Zusammenarbeit

  • README und Dokumentation aktuell halten
  • Releases im GitHub-Tab verwalten
  • Versionierung mit Tags:
git tag v1.0.0
git push origin v1.0.0

💡 9. Empfohlene Projektdateien

Datei Zweck
README.mdProjektbeschreibung & Anleitung
.gitignoreDateien vom Commit ausschließen
LICENSERechtliche Lizenz
CONTRIBUTING.mdAnleitung für Mitwirkende
CODE_OF_CONDUCT.mdVerhaltenskodex
CHANGELOG.mdVersions- & Änderungsverlauf

☁️ 10. Optional: GitHub Actions (CI/CD)

Beispiel für Workflow (.github/workflows/ci.yml):

name: CI
on: [push, pull_request]
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Setup Node.js
        uses: actions/setup-node@v3
        with:
          node-version: '18'
      - run: npm install
      - run: npm test
